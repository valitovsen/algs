<p>Greedy Algorithms:</p>
<ul>

  <li><b>Kruskal's Algorithm</b> - implementation of Kruskal's algorithm to find minimum spanning tree (or forrest) in an undirected weighted graph in <i>O(|E|log|V|)</i> time [<a href="kruskal.py">code</a>][<a href="https://en.wikipedia.org/wiki/Kruskal%27s_algorithm">wiki</a>]</li>
